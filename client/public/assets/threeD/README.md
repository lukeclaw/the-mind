# 3D Lobby Asset Pack

## Textures
- `textures/tech_floor.svg`: sci-fi tile floor pattern.
- `textures/stone_tiles.svg`: neutral stone tile blocks.
- `textures/moss_block.svg`: organic mossy block surface.
- `textures/hazard_stripes.svg`: caution striping for challenge edges.
- `textures/energy_orbs.svg`: glowing decal-style motif.
- `textures/wood_planks.svg`: stylized wood planks.

## Models (OBJ)
- `models/crate_lowpoly.obj`: 1x1x1 crate.
- `models/bench_lowpoly.obj`: simple bench prop, width ~2 units.
- `models/crystal_cluster.obj`: decorative crystal cluster.
- `models/sci_fi_pillar.obj`: pillar/totem landmark.

## Notes
- All assets are intentionally lightweight and stylized placeholders.
- SVG textures can be loaded as standard textures in Three.js (or converted to PNG later).
- OBJ models are untextured geometry intended for `MeshStandardMaterial` color/emissive styling.
